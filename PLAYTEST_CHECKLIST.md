# AI Survivor MVP 플레이테스트 체크리스트

## 빠른 시작

1. Unity 6에서 프로젝트를 엽니다.
2. `Assets/Scenes/SampleScene.unity`를 엽니다.
3. Play를 누릅니다.
4. `WASD` 또는 방향키로 이동합니다.
5. 레벨업 시 클릭 또는 `1`, `2`, `3`으로 강화를 선택합니다.
6. `Esc` 또는 `P`로 일시정지/계속하기를 확인합니다.
7. `Tab`으로 능력치 패널을 숨기거나 다시 표시합니다.
8. `T`로 오디오 테스트 톤을 확인합니다.
9. 게임 오버 후 `R` 또는 `다시 시작` 버튼으로 재시작합니다.

## 5분 기본 점검

- [ ] `SampleScene`에서 Play Mode가 콘솔 오류 없이 시작된다.
- [ ] 첫 60초 동안 기본 적만 등장한다.
- [ ] 1분 즈음 `빠른 적 등장!` 안내가 표시되고 빠른 적이 등장한다.
- [ ] 2분 즈음 `탱커 적 등장!` 안내가 표시되고 탱커 적이 등장한다.
- [ ] 3분 30초 즈음 `적의 공세가 거세집니다!` 안내가 표시된다.
- [ ] 5분 즈음 `정예 적 접근 중!` 안내와 함께 정예 적이 등장한다.
- [ ] 플레이어가 이동, 자동 공격, 처치, 경험치 획득, 레벨업, 강화 선택을 정상 수행한다.
- [ ] 게임 오버 화면에 생존 시간, 최종 레벨, 처치 수, 해금한 무기, 다시 시작 안내가 표시된다.
- [ ] Play Mode 종료 후 Unity 콘솔 오류가 없다.

## 한국어 UI 점검

- [ ] HUD가 `체력`, `레벨`, `경험치`, `시간`을 한국어로 표시한다.
- [ ] 시작 힌트가 이동, 일시정지, 능력치, 오디오 테스트, 강화 선택을 한국어로 안내한다.
- [ ] 일시정지 패널 제목이 `일시정지`로 표시된다.
- [ ] 일시정지 버튼이 `계속하기`, `다시 시작`으로 표시된다.
- [ ] 런타임 능력치 패널 제목이 `능력치`로 표시된다.
- [ ] 레벨업 패널 제목이 `강화 선택`으로 표시된다.
- [ ] 강화 이름/설명이 한국어로 표시되고 텍스트가 넘치지 않는다.
- [ ] 게임 오버 화면이 `게임 오버`, `생존 시간`, `최종 레벨`, `처치 수`, `해금한 무기`, `R 키로 다시 시작`을 표시한다.
- [ ] 1280x720, 1920x1080, 2560x1440, 3840x2160에서 한국어 텍스트가 읽기 쉽다.
- [ ] 한국어 글자가 깨지거나 사각형으로 보이지 않는다. 깨지면 폰트 대응이 필요하다.

## 이동 / 전투 점검

- [ ] `WASD`와 방향키 이동이 정상이다.
- [ ] 대각선 이동이 과도하게 빠르지 않다.
- [ ] 카메라가 플레이어를 자연스럽게 따라간다.
- [ ] 기본 탄이 가장 가까운 적을 자동 조준한다.
- [ ] 발사체가 보이고, 충돌 후 사라진다.
- [ ] 피해 숫자가 적 근처에 크고 선명하게 표시된다.
- [ ] 피해 숫자가 위로 떠오르며 사라진다.
- [ ] 적 HP 바가 피해 후 갱신된다.
- [ ] 탱커 적과 정예 적은 풀 체력에서도 HP 바가 보인다.
- [ ] 기본/빠른 적의 HP 바는 피해를 받은 뒤 나타난다.
- [ ] 피격, 사망, 경험치 획득 피드백이 보인다.

## 웨이브 / 적 종류 점검

- [ ] 기본 적: 빨간색 계열, 균형형, 첫 1분의 유일한 적이다.
- [ ] 빠른 적: 노란색 계열, 작고 빠르며 1분 이후 등장한다.
- [ ] 탱커 적: 보라색 계열, 크고 단단하며 2분 이후 등장한다.
- [ ] 정예 적: 청록색 계열, 탱커보다 크고 단단하며 5분 즈음 1회 등장한다.
- [ ] 각 적 종류가 색과 크기로 구분된다.
- [ ] 정예 적 처치 시 높은 경험치 보상이 체감된다.
- [ ] 정예 적 사망 후 사망 SFX와 경험치 드롭이 정상 동작한다.
- [ ] 풀링된 적이 재사용될 때 색상, 크기, 체력, XP 값, HP 바 상태가 이전 적에서 새지 않는다.

## 경험치 / 강화 점검

- [ ] 적 처치 시 경험치 오브가 드롭된다.
- [ ] 경험치 오브가 가까이 오면 플레이어에게 끌려온다.
- [ ] 경험치 텍스트와 바가 획득 후 갱신된다.
- [ ] 경험치 오브를 획득하면 경험치 바의 파란 채움이 즉시 오른쪽으로 늘어난다.
- [ ] 레벨업 후 남은 경험치 비율에 맞게 경험치 바가 다시 줄어든다.
- [ ] 레벨업 시 게임이 멈추고 3개 강화 선택지가 표시된다.
- [ ] 같은 레벨업 선택지 안에서 중복 강화가 나오지 않는다.
- [ ] `1`, `2`, `3` 단축키와 클릭 선택이 모두 동작한다.
- [ ] 공격력 증가, 공격 속도 증가, 이동 속도 증가가 기존 기능을 유지한다.
- [ ] 최대 체력 증가, 체력 회복, 탄 크기 증가, 경험치 자석, 추가 발사가 기존 기능을 유지한다.
- [ ] 체력 회복을 선택하면 체력 바의 초록 채움이 즉시 늘어난다.
- [ ] 최대 체력 증가를 선택하면 체력 텍스트의 최대값과 체력 바 비율이 함께 갱신된다.
- [ ] 오라 파동 해금과 회전 칼날 해금이 등장하고, 해금 후 다시 등장하지 않는다.
- [ ] 강화 선택 SFX가 재생된다.

## 플레이어 HUD 바 점검

- [ ] 적에게 닿아 피해를 받으면 체력 텍스트와 체력 바가 모두 즉시 감소한다.
- [ ] 체력 바의 초록 채움은 왼쪽에서 오른쪽 방향으로 남은 비율만큼 표시된다.
- [ ] 체력이 0에 가까워지면 체력 바 채움도 거의 사라진다.
- [ ] 체력 회복 후 체력 바가 회복량만큼 다시 늘어난다.
- [ ] 최대 체력 증가 후 현재 체력/최대 체력 비율에 맞게 체력 바 폭이 다시 계산된다.
- [ ] 경험치 획득 후 경험치 텍스트와 경험치 바가 모두 즉시 증가한다.
- [ ] 레벨업 직후 경험치 바가 새 필요 경험치 기준의 남은 경험치 비율로 재계산된다.
- [ ] 씬 재시작 후 체력 바는 가득 찬 상태, 경험치 바는 0 상태로 시작한다.
- [ ] 한국어 HUD 텍스트는 계속 정상 표시된다.

## 무기 호환성 점검

- [ ] 기본 탄은 공격력 증가, 공격 속도 증가, 탄 크기 증가, 추가 발사와 함께 정상 동작한다.
- [ ] 오라 파동이 해금 후 주기적으로 주변 적에게 피해를 준다.
- [ ] 오라 파동 피해에도 피해 숫자와 HP 바 갱신이 보인다.
- [ ] 회전 칼날이 해금 후 플레이어 주변을 돈다.
- [ ] 회전 칼날 접촉 피해에도 피해 숫자와 HP 바 갱신이 보인다.
- [ ] 일시정지, 강화 선택, 게임 오버 중 무기 동작이 멈춘다.

## 일시정지 / 능력치 점검

- [ ] `Esc`와 `P`가 게임플레이 중 일시정지를 토글한다.
- [ ] 일시정지 중 적, 발사체, 타이머, 플레이어 이동이 멈춘다.
- [ ] 일시정지 패널의 전체 능력치가 현재 값과 일치한다.
- [ ] 런타임 능력치 패널이 기본 표시된다.
- [ ] `Tab`으로 런타임 능력치 패널을 숨기고 다시 표시할 수 있다.
- [ ] 능력치에 공격력, 공격 속도, 이동 속도, 체력, 탄 크기, 발사 수, 경험치 흡수 범위가 표시된다.
- [ ] 능력치에 웨이브, 처치 수, 정예 적 상태, 무기 해금 상태가 표시된다.
- [ ] 레벨업 패널이 떠 있을 때 `Esc`, `P`, `Tab`이 강화 선택을 깨지 않는다.
- [ ] 게임 오버 중 `Esc` 또는 `P`로 게임이 재개되지 않는다.

## 결과 화면 / 처치 수 점검

- [ ] 적이 죽을 때마다 처치 수가 증가한다.
- [ ] 처치 수가 런타임 능력치 또는 일시정지 능력치에서 확인된다.
- [ ] 게임 오버 화면의 처치 수가 최종 처치 수와 일치한다.
- [ ] 해금한 무기 목록에 기본 탄이 항상 표시된다.
- [ ] 오라 파동/회전 칼날을 해금한 경우 결과 화면에 표시된다.
- [ ] `R`과 `다시 시작` 버튼 모두 새 런을 시작한다.
- [ ] 재시작 후 오래된 적, 발사체, 경험치 오브, 피해 숫자, HP 바가 남지 않는다.

## 풀링 / 성능 점검

- [ ] 발사체가 반복 사용되어도 크기, 피해, 방향, 수명 상태가 새지 않는다.
- [ ] 경험치 오브가 반복 사용되어도 값과 흡수 상태가 새지 않는다.
- [ ] 피해 숫자가 반복 사용되어도 텍스트, 색, 알파, 위치, 크기가 새지 않는다.
- [ ] 피드백 효과가 반복 사용되어도 색, 크기, 수명이 새지 않는다.
- [ ] 적 풀링이 기본/빠른/탱커/정예 사이에서 안전하게 재설정된다.
- [ ] 많은 적이 있을 때 콘솔 오류나 눈에 띄는 멈춤이 없다.

## 오디오 점검

- [ ] MacBook Speakers에서 `T` 테스트 톤이 들린다.
- [ ] MacBook Speakers에서 발사, 피격, 사망, 경험치, 레벨업, 강화 선택, 플레이어 피해, 게임 오버 SFX가 들린다.
- [ ] AirPods를 Unity 실행 전에 선택한 뒤 `T` 테스트 톤을 확인한다.
- [ ] AirPods에서 일반 SFX가 들리는지 확인한다.
- [ ] macOS 입력 장치를 AirPods가 아닌 장치로 바꾸고 다시 확인한다.
- [ ] Play Mode 중 AirPods 재연결 시 Unity/FMOD 메시지나 무음 문제가 재현되는지 기록한다.
- [ ] AirPods 문제는 MacBook Speakers와 비교해 프로젝트 코드 문제인지 출력 장치 문제인지 구분한다.

## 밸런스 관찰 메모

- [ ] 1분 빠른 적 등장 후 압박이 너무 강하지 않은가?
- [ ] 2분 탱커 적 등장 후 전투가 지루하지 않은가?
- [ ] 3분 30초 공세 강화가 자연스럽게 느껴지는가?
- [ ] 5분 정예 적이 위협적이지만 불공평하지 않은가?
- [ ] 정예 적 HP와 보상이 적절한가?
- [ ] 처치 수가 플레이 목표를 강화하는가?
- [ ] 런타임 능력치 패널이 유용한가, 아니면 나중에 기본 숨김이 나은가?
- [ ] 한국어 텍스트 길이가 UI를 밀거나 겹치게 만들지 않는가?
